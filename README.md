# Stress Testing Frontier AI Systems for Verifiable Reasoning and Code Generation

This repository supports the Zenodo preprint:

**Stress Testing Frontier AI Systems for Verifiable Reasoning and Code Generation: GPT-5.5 Pro / Extra High, Codex, Claude Opus 4.7 Chat, and Claude Code**

Author: Marcel Jiron / MarMarLabs  
Zenodo DOI: https://doi.org/10.5281/zenodo.19779656  
Zenodo record: https://doi.org/10.5281/zenodo.19779656

## Purpose

This repository is intended to make the audit artifacts easier to inspect, reproduce, and critique. It contains prompts, transcripts, test notes, logs, and source-level artifacts used in the paper.

## What is included

- `paper/`: PDF copy of the current preprint.
- `prompts/`: Stress-test prompts and evaluation prompts.
- `artifacts/`: Model outputs, generated files, and code artifacts.
- `verification/`: Test commands, expected results, logs, and manual review notes.
- `metadata/`: Manifest files describing artifact provenance and verification status.

## Important limitation

This is not a controlled laboratory benchmark. It is an artifact-based audit of submitted outputs and verification attempts. Claims should be read together with the available artifacts and logs.
